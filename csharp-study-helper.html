Sessions 2 & 3: Variables, Data Types, Operators

Variables üì¶

  • Think of variables as labeled boxes that store data
  • Must declare TYPE before using: int age = 25;
  • Use var for implicit typing: var name = "John";
  • Naming: Use camelCase, start with letter, no spaces!

Data Types üéØ

  • Value Types: Store actual values (int, float, bool)
  • Reference Types: Store memory addresses (string, object)
  • int: whole numbers (-2B to 2B)
  • double: decimals (15-16 digits precision)
  • bool: true or false only
  • string: text in double quotes

Operators ‚ûï‚ûñ

  • Arithmetic: + - * / % (modulo gives remainder)
  • Comparison: == != < > <= >=
  • Logical: && (AND) || (OR) ! (NOT)
  • Assignment: = += -= *= /=
  • ++ adds 1, -- subtracts 1
  • Order matters! Use () to control precedence

Type Conversion üîÑ

  • Implicit: Automatic (int ‚Üí double)
  • Explicit: Manual with (type) cast
  • Boxing: value type ‚Üí object
  • Unboxing: object ‚Üí value type
  • Convert.ToInt32() for string to int
  • ToString() converts anything to string

Console I/O üíª

  • Console.WriteLine() - prints with new line
  • Console.Write() - prints without new line
  • Console.ReadLine() - reads user input as string
  • Placeholders: {0}, {1}, {2} for formatting
  • Always convert input: Convert.ToInt32(Console.ReadLine())

Quick Tips üí°

  • // for single-line comments
  • /* */ for multi-line comments
  • const for unchanging values
  • Escape sequences: \n (newline) \t (tab)
  • Keywords are RESERVED - can't use as names
  • C# is case-sensitive: age ‚â† Age
